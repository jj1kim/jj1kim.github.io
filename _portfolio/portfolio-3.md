This project focuses on developing image classification models that are optimized for NPU architectures. The work involves model architecture design, quantization techniques, and performance optimization to achieve optimal balance between accuracy and inference speed. The resulting models demonstrate significant improvements in computational efficiency while maintaining competitive classification accuracy.
